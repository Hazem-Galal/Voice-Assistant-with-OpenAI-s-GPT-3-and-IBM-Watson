# VoiceChat — AI Voice Assistant

A browser-based voice assistant that lets you speak or type questions and receive spoken, AI-generated responses. The app is powered by OpenAI's Whisper (speech-to-text), GPT-3.5-Turbo (chat), and TTS (text-to-speech) APIs, served through a lightweight Flask backend.

---

## Features

- **Voice input** — record audio directly in the browser via the microphone button
- **Text input** — type messages in the chat box as an alternative
- **AI responses** — answers generated by GPT-3.5-Turbo and read aloud by OpenAI TTS
- **Multiple voices** — choose from several English, French, and Italian voice options
- **Replay** — re-listen to any previous assistant or user message
- **Light / dark mode** — toggle theme in the UI

## Getting Started

### 1. Clone the repository

```bash
git clone <repo-url>
cd voicechat
```

### 2. Create and activate a virtual environment

```bash
python -m venv .venv

# Windows
.venv\Scripts\activate

# macOS / Linux
source .venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Set your OpenAI API key

Create a `.env` file in the project root:

```env
OPENAI_API_KEY=sk-...
```

The app uses `python-dotenv` and will load this file automatically.

### 5. Run the server

```bash
python server.py
```

Open your browser at [http://localhost:8000](http://localhost:8000).

## API Endpoints

| Method | Path | Description |
|--------|------|-------------|
| `GET` | `/` | Serves the chat UI |
| `POST` | `/speech-to-text` | Accepts raw audio bytes, returns transcribed text |
| `POST` | `/process-message` | Accepts `{ userMessage, voice }`, returns AI text + base64 WAV audio |

## Docker

### Build

```bash
docker build -t voicechat .
```

### Run

```bash
docker run -p 8000:8000 -e OPENAI_API_KEY=sk-... voicechat
```

> **Note:** The Dockerfile copies certificates from `certs/` and sets `OPENAI_API_KEY=skills-network` for lab environments. Remove or adjust those lines before deploying to production.
